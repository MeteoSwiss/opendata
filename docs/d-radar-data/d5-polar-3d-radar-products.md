# Polar 3D radar products

Single site volume data of [MeteoSwiss' weather radar](https://www.meteoswiss.admin.ch/weather/measurement-systems/atmosphere/weather-radar-network.html) are available via the 'Open Radar Data (ODR)' API, run by EUMETNET's [MeteoGate](https://meteogate.eu/).

- [What is the 'Open Radar Data (ODR)' API](https://eumetnet.github.io/openradardata-documentation/)
- [Open Radar Data API (Swagger)](https://api.meteogate.eu/eu-eumetnet-weather-radar/docs)
- [How to download weather radar data](https://eumetnet.github.io/openradardata-documentation/2-ORD-API-discovering-and-accessing-data/)
