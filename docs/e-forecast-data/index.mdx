# Forecast data

| No.    | Dataset/Collection | Data availability |
|:------:|:---------|:---------|
| E1     | [Short-term forecast data](/e-forecast-data/e1-short-term-forecast-data) <br></br> Forecast of temperature and precipitation with 1 km grid size for the next few minutes up to a maximum of 6 hours ahead <br></br> Calculated every 5-10 minutes, available for the last 6 hours <br></br> | Data [on request](https://www.meteoswiss.admin.ch/about-us/contact/contact-form.html) |
| **E2** | [**Numerical weather forecasting model ICON-CH1-EPS**](/e-forecast-data/e2-e3-numerical-weather-forecasting-model) <br></br> Forecast of temperature, humidity, precipitation, wind, pressure, geopotential, evaporation and radiation with 1 km grid size for the next 33 hours <br></br> Updated every 3 hours, available for the last 24 hours <br></br> :information_source: [**Changelog**](/e-forecast-data/e2-e3-numerical-weather-forecasting-model-changelog) | [**Download options**](/e-forecast-data/e2-e3-numerical-weather-forecasting-model#download-options) |
| **E3** | [**Numerical weather forecasting model ICON-CH2-EPS**](/e-forecast-data/e2-e3-numerical-weather-forecasting-model) <br></br> Forecast of temperature, humidity, precipitation, wind, pressure, geopotential, evaporation and radiation with 2.1 km grid size for the next 120 hours <br></br> Updated every 6 hours, available for the last 24 hours <br></br> :information_source: [**Changelog**](/e-forecast-data/e2-e3-numerical-weather-forecasting-model-changelog) | [**Download options**](/e-forecast-data/e2-e3-numerical-weather-forecasting-model#download-options) |
| **E4** | [**Local forecast data**](/e-forecast-data/e4-local-forecast-data) <br></br>  <br></br> | [**Data download**](/e-forecast-data/e4-local-forecast-data) |
