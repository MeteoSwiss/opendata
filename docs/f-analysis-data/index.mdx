# Analysis data

| No.    | Dataset/Collection | Data availability |
|:------:|:---------|:---------|
| **F1** | [Numerical weather analysis data](/f-analysis-data/f1-numerical-weather-analysis) <br></br> Analysis of temperature, humidity, wind, pressure, radiation, pollen concentration and other meteorological variables at 1 km grid resolution, generated by the KENDA-CH1 data assimilation system. The dataset also includes KENDA-CH1 1-hour First Guess estimates for precipitation, radiation and other variables. Updated hourly and are available for the last 24 hours.  <br></br> |  [**Download options**](/e-forecast-data/e2-e3-numerical-weather-forecasting-model#download-options) || **E2** | [**Numerical weather forecasting model ICON-CH1-EPS**](/e-forecast-data/e2-e3-numerical-weather-forecasting-model) <br></br> Forecast of temperature, humidity, precipitation, wind, pressure, geopotential, evaporation and radiation with 1 km grid size for the next 33 hours <br></br> Updated every 3 hours, available for the last 24 hours <br></br> | [**Download options**](/f-analysis-data/f1-numerical-weather-analysis#download-options) |
