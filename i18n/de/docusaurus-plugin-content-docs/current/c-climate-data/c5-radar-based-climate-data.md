# Räumliche Klimaanalysen - Hagel

Die [Hagelklimatologie](https://www.meteoschweiz.admin.ch/klima/klima-der-schweiz/hagelklimatologie.html) wurde im Rahmen einer öffentlich-privaten Partnerschaft entwickelt und bildet eine landesweit einheitliche Grundlage für die Risikobewertung von Hagelereignissen. Diese Gitterdatenprodukte werden aus Radardaten abgeleitet, die unter anderem [mithilfe des Resampling-Verfahrens HailStoRe für statistische Auswertungen weiterverarbeitet](https://www.meteoswiss.admin.ch/climate/the-climate-of-switzerland/hail-climatology/data-and-methods.html) wurden (weitere Informationen finden Sie in den verlinkten detaillierten Produktdokumentationen). Die Hageldatensätze werden ausschliesslich in der Hagelsaison (April – September) berechnet.

[Räumliche Klimaanalysen](https://www.meteoschweiz.admin.ch/klima/klima-der-schweiz/raeumliche-klimaanalysen.html) sind Datensätze, welche die geografische Verteilung und zeitliche Entwicklung von bodennahen Klimavariablen beschreiben. Die Daten werden räumlich umfassend auf einem Gitter mit 1 km Rasterweite dargestellt und decken die gesamte Schweiz ab. Einige Parameter reichen mehrere Jahrzehnte bis über ein Jahrhundert zurück und werden kontinuierlich in nahezu Echtzeit aktualisiert. Die Datenwerte werden statistisch aus Messungen von Wetterstationen, Satelliten und Radaren geschätzt und können als Pseudo-Beobachtungen der Bedingungen im jeweiligen Gitterpixel verstanden werden.

MeteoSchweiz bietet eine breite Palette an vorgefertigten räumlichen Klimaanalysen an. Eine umfassende Liste der Datenprodukte und detaillierte Dokumentationen finden Sie auf unserer Webseite über [Räumliche Klimaanalysen](https://www.meteoschweiz.admin.ch/klima/klima-der-schweiz/raeumliche-klimaanalysen.html). Eine kompakte Übersicht über andere räumliche Klimaanalysen als Hagel finden Sie hier: [Klimadaten (C3, C4)](/c-climate-data/c3-ground-based-climate-data)

:::info

Die Hageldatensätze werden fortlaufend aktualisiert: 
- Monatliche Aktualisierungen sind jeweils am Ende des Folgemonats verfügbar
- Jährliche Aktualisierungen sind jeweils Ende Oktober verfügbar
- 'Hagelgefährdung – Wiederkehrwerte' sind statische Informationen 

Die Dateien des aktuellen Monats/Jahres sind in der **'Item'-Liste** verfügbar (siehe [STAC Browser](https://data.geo.admin.ch/browser/index.html#/collections/ch.meteoschweiz.ogd-radar-derived-grid). Ältere sind im 'Item' `CH-Archiv` abgelegt. Gefährdungskarten / Wiederkehrwerte sind ebenfalls im 'Item' `CH-Archiv` abgelegt.

:::

## Daten herunterladen {#data-download}

Die 'Open Data' von MeteoSchweiz dürfen uneingeschränkt weiterverwendet werden; bei der Wiedergabe oder Weiterverbreitung der Daten ist **die Quelle anzugeben** ("**Quelle: MeteoSchweiz**").

:white_check_mark: Mit der Verwendung von 'Open Data' von MeteoSchweiz bestätigen Sie die Kenntnisnahme der [Nutzungsbedingungen](/general/terms-of-use).

Der DOI des Hagelklimadatensatzes lautet: [10.18751/Climate/Griddata/CHHC/1.0](https://doi.org/10.18751/Climate/Griddata/CHHC/1.0)

### Daten automatisch herunterladen

Laden Sie **Dateien** automatisch über die BGDI REST-API herunter: [`https://data.geo.admin.ch/api/stac/v1/collections/ch.meteoschweiz.ogd-radar-derived-grid`](https://data.geo.admin.ch/api/stac/v1/collections/ch.meteoschweiz.ogd-radar-derived-grid)

Der STAC-Browser kann ein nützliches Hilfsmittel sein, um die Verwendung der API zu erleichtern [`https://data.geo.admin.ch/browser/index.html#/collections/ch.meteoschweiz.ogd-radar-derived-grid`](https://data.geo.admin.ch/browser/index.html#/collections/ch.meteoschweiz.ogd-radar-derived-grid)

### Daten manuell herunterladen

Wählen Sie **Dateien** manuell über den BGDI [STAC Browser](https://data.geo.admin.ch/browser/index.html#/collections/ch.meteoschweiz.ogd-radar-derived-grid) aus und laden Sie sie herunter.

⚠️ **Bitte beachten Sie die Informationen im Infokasten oben, um zu sehen, in welchen 'Items' welche Dateien abgelegt sind.**


## Datenformat

Das Datenformat ist [`NetCDF`](https://www.unidata.ucar.edu/software/netcdf/).

## Koordinatensystem

Das Koordinatensystem ist [`Swiss LV95`](https://www.swisstopo.admin.ch/de/schweizer-koordinatensystem) / [`EPSG:2056`](https://epsg.io/2056).
