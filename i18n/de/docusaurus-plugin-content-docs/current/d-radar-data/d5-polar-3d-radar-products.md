# Polar-3D-Radarprodukte

Volumendaten einzelner Standorte des [Schweizer Wetterradarnetzes](https://www.meteoschweiz.admin.ch/wetter/messsysteme/atmosphaere/das-schweizer-wetterradarnetz.html) von MeteoSchweiz sind über die 'Open Radar Data (ODR)'-API verfügbar, die von [EUMETNET MeteoGate](https://meteogate.eu/) betrieben wird.

- [Was ist die 'Open Radar Data (ODR)'-API](https://eumetnet.github.io/openradardata-documentation/) 
- [Open Radar Data API (Swagger)](https://api.meteogate.eu/eu-eumetnet-weather-radar/docs) 
- [So laden Sie Wetterradardaten herunter](https://eumetnet.github.io/openradardata-documentation/2-ORD-API-discovering-and-accessing-data/)
