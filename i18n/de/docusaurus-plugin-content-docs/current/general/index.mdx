# Allgemeine Informationen

Willkommen auf der Open-Data-Dokumentation von MeteoSchweiz.

[MeteoSchweiz](https://www.meteoschweiz.admin.ch/ueber-uns/gesetzlicher-auftrag.html) macht ihre Daten schrittweise als [Open Government Data](https://www.bfs.admin.ch/bfs/de/home/dienstleistungen/ogd/kurz-erklart.html) (OGD) zugänglich.

## Offene Wetter- und Klimadaten

Die verfügbaren Daten können entweder manuell heruntergeladen oder automatisch als Dateien bezogen werden:
- [**A - Bodenmessdaten**](/a-data-groundbased)
- [**B - Atmosphärenmessdaten**](/b-data-atmosphere)
- [**C - Klimadaten**](/c-climate-data)
- [**D - Radardaten**](/d-radar-data)
- [**E - Prognosedaten**](/e-forecast-data)

Weitere Datensätze werden in den folgenden Monaten veröffentlicht. Sobald ein Releasetermin feststeht, wird das voraussichtliche Quartal auf den Übersichtsseiten oben (A bis E) bekannt gegeben.

Wir arbeiten zudem daran, bestimmte Daten mittels individueller API-Abfragen abrufbar zu machen. Diese Option wird allerdings nicht vor Ende 2026 verfügbar sein.

Auf der [Roadmap unseres Open-Data-Dienstes](/general/roadmap) sehen Sie, woran wir gerade arbeiten und bis wann.

## Nutzungsbedingungen

Die 'Open Data' von MeteoSchweiz dürfen uneingeschränkt weiterverwendet werden; bei der Wiedergabe oder Weiterverbreitung der Daten ist **die Quelle anzugeben** ("**Quelle: MeteoSchweiz**").

:white_check_mark: Mit der Verwendung von 'Open Data' von MeteoSchweiz bestätigen Sie die Kenntnisnahme der [Nutzungsbedingungen](/general/terms-of-use).
