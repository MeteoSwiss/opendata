# Informazioni generali

Benvenuti alla documentazione Open Data di MeteoSvizzera.

A partire da maggio 2025, [MeteoSvizzera](https://www.meteosvizzera.admin.ch/chi-siamo/mandato-legale.html) renderà gradualmente accessibili i suoi dati come [Open Government Data](https://www.bfs.admin.ch/bfs/it/home/servizi/ogd/in-breve.html) (OGD). I dati disponibili come file possono essere scaricati manualmente o ottenuti in modalità automatica.

Stiamo anche lavorando per consentire l’accesso ad alcuni dati attraverso query API individuali. Tuttavia, questa opzione non sarà disponibile prima del 2026.

## Fase beta pubblica – aprile/maggio 2025

Stiamo attualmente configurando il nostro servizio in versione beta.

:::warning

Durante questa fase, possono essere apportate modifiche in qualsiasi momento senza preavviso.

:::

È già possibile accedere ai Open Data disponibili tramite l'[API STAC di geo.admin.ch](/general/download). Leggere le [Condizioni di utilizzo dell’infrastruttura](/general/terms-of-use#4-utilizzo-dellinfrastruttura).

Gli URL di accesso sono disponibili nella documentazione relativa ai set di dati/'collections' corrispondenti:
- [**A – Dati delle misurazioni al suolo**](/a-data-groundbased)
- [**C – Dati climatici:** Serie omogenee di misurazioni](/c-climate-data)

I dati del modello numerico di previsione meteorologica sono messi a disposizione per il download dal [CSCS](https://www.cscs.ch):
- [**E – Dati di previsione:** Modello numerico di previsione meteorologica ICON-CH1-EPS](/e-forecast-data)

:::tip

Saremo lieti di ricevere [il vostro feedback](/general/contact). Durante la fase beta pubblica, potremmo non rispondere alle vostre domande singolarmente, ma in forma aggregata tramite il nostro [repository di Open Data (GitHub)](https://github.com/MeteoSwiss/opendata/discussions/categories/public-beta-april-2025).

:::


## Condizioni di utilizzo

I 'Open Data' di MeteoSvizzera *possono essere riutilizzati senza restrizioni; in caso di riproduzione o diffusione dei dati è necessario **citare la fonte** (“**Fonte: MeteoSvizzera**”).*

:white_check_mark: Utilizzando i 'Open Data' di MeteoSvizzera, confermi di aver preso visione delle [Condizioni di utilizzo](/general/terms-of-use).
