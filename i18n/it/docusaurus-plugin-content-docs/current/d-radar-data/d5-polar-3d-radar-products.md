# Prodotti radar polari 3D

I dati volumetrici relativi alle singole stazioni della [rete svizzera di radar meteorologici](https://www.meteosvizzera.admin.ch/tempo/i-sistemi-di-rilevamento/atmosfera/la-rete-svizzera-di-radar-meteorologici.html) di MeteoSvizzera sono disponibili tramite l'API 'Open Radar Data (ODR)', gestita da [EUMETNET MeteoGate](https://meteogate.eu/).

- [Cos'è l'API 'Open Radar Data (ODR)'](https://eumetnet.github.io/openradardata-documentation/)
- [Open Radar Data API (Swagger)](https://api.meteogate.eu/eu-eumetnet-weather-radar/docs)
- [Come scaricare i dati dei radar meteorologici](https://eumetnet.github.io/openradardata-documentation/2-ORD-API-discovering-and-accessing-data/)
