# Informations générales

Bienvenue sur la documentation Open Data de MétéoSuisse.

A partir du 22 mai 2025, [MétéoSuisse](https://www.meteosuisse.admin.ch/portrait/mandat-legal.html) rend progressivement ses données accessibles en tant qu'[Open Government Data](https://www.bfs.admin.ch/bfs/fr/home/services/ogd/en-bref.html) (OGD). Les données disponibles peuvent être téléchargées manuellement ou obtenues automatiquement sous forme de fichiers.

Nous commençons par
- [**A – Données de mesures au sol**](/a-data-groundbased)
- [**C – Données climatiques :** Séries de mesures homogènes](/c-climate-data)
- [**E – Données de prévision :** Modèle numérique de prévision météorologique ICON-CH1/2-EPS](/e-forecast-data)

D'autres jeux de données seront publiés dans les mois suivants. Le trimestre prévu pour chaque ensemble de données sera annoncé sur notre [page d'accueil Open Data](https://www.meteosuisse.admin.ch/services-et-publications/service/open-data.html).

Nous travaillons également à permettre l'accès à certaines données par des requêtes API individuelles. Toutefois, cette option ne sera pas disponible avant 2026.


## Conditions d'utilisation

Les 'Open Data' de MétéoSuisse peuvent être réutilisées sans restriction; **la source doit être indiquée** lors de la reproduction ou de la diffusion des données ("**Source : MétéoSuisse**").

:white_check_mark: En utilisant les 'Open Data' de MétéoSuisse, vous confirmez avoir pris connaissance des [Conditions d'utilisation](/general/terms-of-use).
