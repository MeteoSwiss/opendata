# Informations générales

Bienvenue sur la documentation Open Data de MétéoSuisse.

A partir de Mai 2025, [MétéoSuisse](https://www.meteosuisse.admin.ch/portrait/mandat-legal.html) rend progressivement ses données accessibles en tant qu'[Open Government Data](https://www.bfs.admin.ch/bfs/fr/home/services/ogd/en-bref.html) (OGD). Les données disponibles peuvent être téléchargées manuellement ou obtenues automatiquement sous forme de fichiers.

Nous travaillons également à permettre l'accès à certaines données par des requêtes API individuelles. Toutefois, cette option ne sera pas disponible avant 2026.


## Phase bêta publique – avril/mai 2025

Nous sommes actuellement en train de mettre en place notre service en version bêta.

:::warning

Pendant cette phase, des modifications peuvent être apportées à tout moment sans préavis.

:::

Vous pouvez déjà accéder aux données ouvertes disponibles via l'[API STAC de geo.admin.ch](/general/download). Veuillez lire les [conditions d'utilisation de l'infrastructure](https://opendatadocs.meteoswiss.ch/general/terms-of-use#4-utilisation-de-l-infrastructure).

Vous trouverez les URL d'accès dans la documentation relative aux jeux de données/'collections' correspondants :
- [A – Données de mesures au sol](/a-data-groundbased)
- [C – Données climatiques : Séries de mesures homogènes](/c-climate-data)

Les données du modèle numérique de prévision météorologique sont mises à disposition pour téléchargement par le [CSCS](https://www.cscs.ch) :
- [E – Données de prévision : Modèle numérique de prévision météorologique ICON-CH1/2-EPS](/e-forecast-data)

:::tip

Nous serions ravis de recevoir [vos commentaires](/general/contact). Pendant notre phase bêta publique, nous ne répondrons peut-être pas à vos questions individuellement, mais de manière groupée via notre [référentiel de l'Open Data (GitHub)](https://github.com/MeteoSwiss/opendata/discussions/categories/public-beta-april-2025).

:::


## Conditions d'utilisation

Les 'Open Data' de MétéoSuisse peuvent être réutilisées sans restriction; **la source doit être indiquée** lors de la reproduction ou de la diffusion des données ("**Source : MétéoSuisse**").

Consultez nos [conditions d'utilisation des 'Open Data'](/general/terms-of-use) pour connaître les licences et les conditions d'utilisation des données.
