# Produits radar polaires 3D

Les données volumiques provenant de sites spécifiques du [réseau suisse de radars météorologiques](https://www.meteosuisse.admin.ch/meteo/systemes-de-mesure/atmosphere/le-reseau-suisse-de-radars-meteorologiques.html) de MétéoSuisse sont disponibles via l'API 'Open Radar Data (ODR)', gérée par [EUMETNET MeteoGate](https://meteogate.eu/).

- [Qu'est-ce que l'API 'Open Radar Data (ODR)'](https://eumetnet.github.io/openradardata-documentation/)
- [Open Radar Data API (Swagger)](https://api.meteogate.eu/eu-eumetnet-weather-radar/docs)
- [Comment télécharger des données radar météorologiques](https://eumetnet.github.io/openradardata-documentation/2-ORD-API-discovering-and-accessing-data/)
